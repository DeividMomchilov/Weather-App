# Времето в България

Просто уеб приложение, което показва текущото време за всяко населено място в България.

# Функционалност
- Търсене по име на град (на български)
- Показва текуща температура и условия
- Използва данни от Open-Meteo и OpenStreetMap

# Използвани технологии
- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo API
- Nominatim API

# Как да стартирате приложението
1. Изтеглете файловете index.html, styles.css и index.js в една и съща папка.
2. Отворете index.html с браузър.
3. Въведете име на град и натиснете **„Търсене“**.
